# Android Launcher for ADD Support

## Project Overview
This Android launcher is designed specifically to enhance the user experience for individuals with Attention Deficit Disorder (ADD). It focuses on task and event management to promote organization and productivity.

## Features
- **Customizable Home Screen**: Users can arrange apps and widgets according to their preferences, making it easier to access frequently used tools.
- **Task Management**: Integrates a task manager that allows users to create, manage, and prioritize tasks.
- **Event Notifications**: Provides reminders and notifications for events to help users stay on track.
- **Focus Modes**: Users can activate focus modes that limit distractions by hiding unnecessary apps and features.
- **Customization Options**: Offers various themes and layouts to accommodate individual user needs.

## Installation
To install the launcher, follow these steps:
1. Download the APK from the releases section.
2. Enable installations from unknown sources in your device settings.
3. Open the downloaded APK and follow the installation prompts.

## Usage
After installation:
1. Set the launcher as your default home screen.
2. Explore settings to customize your experience based on your needs.
3. Use the task management feature to organize daily tasks.
